\# FF14 仙人微彩計算器 (Mini Cactpot Solver)



\*\*FF14 仙人微彩計算器\*\* 是一款專為《Final Fantasy XIV》(FFXIV) 金碟遊樂場小遊戲「仙人微彩 (Mini Cactpot)」設計的網頁版輔助工具。透過機率分析與期望值計算，幫助玩家每日獲得最高 MGP 獎勵。



👉 \*\*立即使用：\[點擊開啟計算器](https://rarakumoai.github.io/cactpot-solver/)\*\*



\## ✨ 主要功能

\* \*\*期望值計算\*\*：自動計算所有路徑的期望值，標示最佳選擇。

\* \*\*多國語言支援\*\*：繁體中文、簡體中文、英文、日文、德文、法文、韓文。

\* \*\*跨平台支援\*\*：支援 iOS (Safari) 與 Android (Chrome) 安裝為 PWA 應用程式。

\* \*\*無限填數字模式\*\*：方便進行測試或模擬不同盤面。



\## 📱 安裝教學

本工具支援 PWA (Progressive Web App) 技術，可直接安裝於手機主畫面，體驗如同原生 App。

\* \*\*iOS\*\*: 使用 Safari 開啟 -> 分享 -> 加入主畫面。

\* \*\*Android\*\*: 使用 Chrome 開啟 -> 選單 -> 加到主畫面。
