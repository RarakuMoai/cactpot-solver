# FF14 仙人微彩計算器 (Mini Cactpot Solver)

**FF14 仙人微彩計算器** 是一款專為《Final Fantasy XIV》(FFXIV) 金碟遊樂場小遊戲「仙人微彩 (Mini Cactpot)」設計的網頁版輔助工具。透過機率分析與期望值計算，幫助玩家每日獲得最高 MGP 獎勵。

👉 **立即使用：[點擊開啟計算器](https://rarakumoai.github.io/cactpot-solver/)**

## ✨ 主要功能
* **期望值計算**：自動計算所有路徑的期望值，並標示最佳選擇。
* **多國語言支援**：支援繁體中文、簡體中文、English、日本語、Deutsch、Français、한국어。
* **跨平台 PWA 支援**：
    * **iOS**: 支援加入主畫面，體驗如原生 App。
    * **Android**: 支援安裝應用程式，或加入主畫面捷徑。
* **無限填數字模式**：解除鎖定限制，方便進行測試或模擬不同盤面。

## 📱 安裝教學
本工具採用 PWA (Progressive Web App) 技術，可直接安裝於手機主畫面。

### iOS (iPhone/iPad)
1. 使用 **Safari** 瀏覽器開啟。
2. 點擊下方工具列的 **【分享】** 按鈕。
3. 選擇選單中的 **「加入主畫面」**。

### Android
1. 使用 **Chrome** 瀏覽器開啟。
2. 點擊右上角選單 **【⋮】**。
3. 選擇 **「加到主畫面」** 或 **「安裝應用程式」**。
